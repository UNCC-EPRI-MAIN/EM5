Detailed Description

This class is the firmware used to control the wheel motors via the Sabertooth2x60 hardware.

A PWM signal is passed throug the low pass filter, converting it to an analog signal. The Sabertooth2x60 then uses the signal to control a motor. This class controls only a single motor via one PWM signal.
