Detailed Description

Pin assignments for Raspberry Pi GPIO.

This file is an auxillary file used to assign GPIO pin assignments for various purposes. This file ensures pin numbers do not conflict and provides a mechanism for quick changes. All pin numbers use BCM mode. See Raspberry Pi GPIO for more information.
