Detailed Description

This class is the firmware used to control the blade motor driver.

A PWM signal is sent to the MD30C to indicate speed of blade motors. One signal is used for all three blade motors. Note: Blades will not spin if power is cut off by an open relay Blades are to be contolled by higher module -> BladeControl
